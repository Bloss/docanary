## 安装 Docanary

请确保安装最新版的 [Node](https://nodejs.org/en/download/), 和 [npm](https://www.npmjs.com/)

```bash
npm i docanary -g
```

## 初始化你的项目

```bash
docanary init <rootDirectory>
```

`rootDirectory` 默认是当前目录.

例子:

```bash
docanary init docanary-example
```
